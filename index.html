Office Web Controls Reference

These sections explains the elements that make up the Office Web Controls Suite.

An Office Web Controls applications can interact with HTML, CSS and the Document Object Model (DOM), which represents HTML and browser objects.

  • For more information about interacting with HTML elements, see Interacting with HTML elements.

  • For more information about CSS, see Interacting with Cascading Style Sheets.

  • For more information about the DOM, see Interacting with the Document Object Model (DOM).

Important

The Office Web Controls Suite is an application which is built on top of AngularJS which means that a good knowledge of JavaScript is needed when you want to modify the source code of project.

Also, when you want to install the Office Web Controls Suite you have 2 options, and that's installing from source or installing from a release package.
