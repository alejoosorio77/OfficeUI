OfficeUI Reference

Welcome to the Office User Interface, further called OfficeUI. We've got everything you need to create great Web Applications. Create a simple JSon file and recreate the User Interface like Microsoft is using in their applications such as Microsoft Word, Microsoft Outlook, Microsoft Excel and many others. OfficeUI moves your application to a higher level by creating a rich, intuitive user interface that every user will understand.
